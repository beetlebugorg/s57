# Quick Start Example

Basic S-57 chart parsing example that demonstrates:
- Creating a parser
- Parsing a chart file
- Accessing chart metadata
- Getting chart bounds

## Running

```bash
go run main.go
```

## Expected Output

```
Chart: US5MA22M
Edition: 3
Features: 1234
Bounds: [-71.1000,42.3000] to [-71.0000,42.4000]
```

(Actual values depend on your chart file)
